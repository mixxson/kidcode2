# Platforma do nauki programowania dla dzieci

To jest szkielet projektu "platforma do nauki programowania dla dzieci".
Struktura projektu zawiera backend (Node.js + Express) i frontend (React + Vite).

Celem tej wersji startowej jest szybkie uruchomienie lokalnego środowiska i dalsza rozbudowa.

Jak używać

1. Backend:
   - Przejdź do `backend` i uruchom `npm install`, a potem `npm run dev`.
2. Frontend:
   - Przejdź do `frontend`, uruchom `npm install`, a potem `npm run dev`.

Jeśli chcesz, mogę zainicjować repozytorium git i dodać pierwszy commit oraz przygotować pliki Docker.
