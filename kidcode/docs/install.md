# Instalacja i uruchomienie projektu (PL)

1. Backend

```powershell
cd d:\kidcode2\kidcode\backend
npm install
npm run dev
```

Serwer będzie dostępny pod `http://localhost:4000/api`.

2. Frontend

```powershell
cd d:\kidcode2\kidcode\frontend
npm install
npm run dev
```

Aplikacja frontendowa działa zwykle na `http://localhost:5173`.

Uwaga: Upewnij się, że backend działa przed uruchomieniem frontendu (CORS i adresy API).
